# 21.JavaScript语法小技巧

### 21.1 默认参数赋值

冗长写法：
```JavaScript
function fun(param) {
    if(param == null) {
        param = {name:'tiger'};
    }
}
```
简洁写法：
```JavaScript
function fun(param) {
    param = param || {name:'tiger'};
}
```

### 21.2 回调判空执行

冗长写法：
```JavaScript
function fun(callback) {
    if(callback) {
        callback();
    }
}
```

简洁写法：

```JavaScript
function fun(callback) {
    callback && callback();
}
```
